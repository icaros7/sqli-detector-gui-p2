## 환경
- 대상 플렛폼: `Microsoft Windows 10 혹은 그 이상` 64비트 (윈도우 7 환경에서는 테스트되지 않음)
- 대상 .Net SDK: `7.0`
- 지원 브라우저:   
				- `Edge`

				- `Firefox`
